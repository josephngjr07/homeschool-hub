# Homeschool Hub

A calm planning tool for newcomer homeschooling parents. It acts as a trusted
"external brain" built around a weekly rhythm — capture ideas, plan the week,
see today, check it off — so the parent depends on it the way they depend on a
calendar.

## Language

**Resource**:
A saved educational idea or link, captured so it can later be planned into a Task.
A Resource exists to serve the plan — it is not a permanently-browsed collection.
_Avoid_: Bookmark, pin, content, material

**Inbox**:
The holding area where captured Resources wait to be turned into Tasks. It is
meant to *drain* (be processed into the plan), not to accumulate as a library.
_Avoid_: Vault, library, organizer, collection

**Task**:
A planned learning activity for a specific day, assigned to one or more Children
(or the whole family), checked off when done. The unit the plan is built from.
A Task may link back to the Resource it came from, so it can be opened while
doing it. Repeats are many independent Tasks, never a recurrence rule.
_Avoid_: Lesson, assignment, event, todo

**Child**:
A kid the parent homeschools. In the app a Child is primarily a tag/filter on
Tasks (name + color), not a profile to manage.
_Avoid_: Student, kid, pupil

**Today**:
The single calm view answering "what are we doing today?" — the whole family's
Tasks for the current day, optionally filtered to one Child.
_Avoid_: Agenda, feed, home

**Starter week**:
A pre-filled first week generated from the parent's kids + chosen subjects during
a two-step setup, so the parent never faces an empty planner.
_Avoid_: Template, sample, demo data, seed
